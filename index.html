    This is an analysis and visualization of weather of cities across the world. The main objective was to analyse the relationship of various factors like temperature, humidity, cloudiness and wind speed with respect to the distance from the equator.
    For this analysis, approx. 1700 coordinate pairs of latitude and longitude were created randomly and citypy (a python library) was used to generate city names for respective coordinates. After selecting unique non-repeat observations Of the 1500 cities randomly selected we use weather API to pull data for those random cities in a data frame using jupyter notebook. cities, a weather check was performed on each of the cities using a series of successive API calls from OpenWeatherMap. Then all different point of interests were plotted against latitude using Matplotlib and Seaborn libraries.
